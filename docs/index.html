Quickstart

Hızlı Başlangıç

Install the recommended profile (~450 skills) or a specific category — CLI or GUI.

Önerilen profili (~450 beceri) veya belirli bir kategoriyi yükleyin — CLI veya GUI.

Python 3.9+ 
# Clone the project
❮ git clone https://github.com/bayraktarozcan/AgentSynapse.git
❮ cd AgentSynapse

# Recommended install — 450+ skills, auto-curated (34 repos)
❮ python skills.py

# Launch graphical interface (tkinter)
❮ python skills.py --gui

# Full trusted suite — all K1-K8 repos (39 repos, ~600 skills)
❮ python skills.py trusted

# Install a specific category
❮ python skills.py K2        # AI & LLM skills
❮ python skills.py K3        # Cloud & Backend skills

# Include community and risk repos
❮ python skills.py all        # All repos including K9-K10 (48 repos)

# Output
✔ 459 skills installed → ~\.agents\skills\
Python 3.9+ 
# Turkish aliases also work:
❮ python skills.py onerilen    # Önerilen profil (varsayılan)
❮ python skills.py guvenli     # Güvenli profil (K1-K8)
❮ python skills.py tumu        # Tümü (K1-K10)
❮ python skills.py --lang tr   # Türkçe arayüz

Frequently Asked Questions

Sıkça Sorulan Sorular

Common questions about Agent Skills and the ecosystem.

Agent Becerileri ve ekosistemi hakkında sık sorulan sorular.

SKILL.md is a standardized format for AI agent instructions. Tools like Claude Code, OpenCode, and Cursor natively discover and load these files, giving agents domain expertise on demand.

SKILL.md, AI ajan talimatları için standartlaştırılmış bir formattır. Claude Code, OpenCode ve Cursor gibi araçlar bu dosyaları yerel olarak keşfeder ve yükler.

Skills install to ~/.agents/skills/. Claude Code reads this directory automatically. For OpenCode, configure OPENCODE_SKILLS_DIR. Most modern agent tools scan this path.

Beceriler ~/.agents/skills/ klasörüne kurulur. Claude Code bu dizini otomatik okur. OpenCode için OPENCODE_SKILLS_DIR yapılandırılabilir.

Yes! Submit a pull request adding your repo to repos.json with a category assignment. Core and AI categories require Community review; K9-K10 have lower thresholds.

Evet! Deponuzu repos.json'a kategori atamasıyla ekleyerek bir PR gönderin.

First run clones 34-48 repos (1-5 min depending on connection). Subsequent runs skip cloning and process in seconds. Dedup and categorization are near-instant.

İlk çalıştırma 34-48 depoyu klonlar (bağlantıya bağlı 1-5 dk). Sonraki çalıştırmalar saniyeler sürer.

Recommended (default, 34 repos): highest quality, curated. Trusted (K1-K8, 39 repos): full suite without opt-in categories. All (48 repos): includes Community and Risk repos.
